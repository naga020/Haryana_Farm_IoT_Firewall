# Haryana Farm IoT Adaptive Firewall

AI-based anomaly-detection firewall for protecting farm IoT sensors using Isolation Forest.
Project files:
- farm_iot_traffic.csv – simulated sensor traffic
- farm_firewall_complete.csv – traffic + anomaly scores
- threats_detected.csv – only packets flagged as threats
